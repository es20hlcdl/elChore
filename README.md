# elChore

Storymap editorial inspirado en la reserva El Chore, construido con Node.js sin dependencias externas.

## Ejecutar

```bash
npm run dev
```

o

```bash
npm start
```

Luego abre `http://localhost:3000`.

## Estructura

- `server.mjs`: servidor HTTP simple en Node.js
- `public/index.html`: estructura del storymap
- `public/styles.css`: estilos y composición visual
- `public/app.js`: animaciones, línea temporal y comportamiento del mapa

## Siguiente paso sugerido
